# USER GUIDE Messagerie

## Utilisation de base

### Accéder au webmail (SOGo)

1. Ouvrir un navigateur web
2. Aller à : https://192.168.10.35/SOGo/
3. Accepter le certificat de sécurité si demandé
4. Entrer vos identifiants :
   - **Nom d'utilisateur** : votre adresse email complète (ex : petra.rossi@tssr.lan)
   - **Mot de passe** : votre mot de passe
5. Cliquer sur **Connexion**

---

### Lire ses emails

1. Dans SOGo, les emails reçus s'affichent dans la **Boîte de réception**
2. Cliquer sur un email pour le lire

---

### Envoyer un email

1. Cliquer sur **Rédiger** ou **Nouveau message**
2. Remplir les champs :
   - **À** : adresse email du destinataire
   - **Objet** : sujet du message
   - **Corps** : contenu du message
3. Cliquer sur **Envoyer**


---

### Répondre à un email

1. Ouvrir l'email
2. Cliquer sur **Répondre**
3. Rédiger la réponse
4. Cliquer sur **Envoyer**

---

### Supprimer un email

1. Sélectionner l'email
2. Cliquer sur **Supprimer** (icône corbeille)
3. L'email est déplacé dans la **Corbeille**

---

## Utilisation avancée

### Ajouter une pièce jointe

1. Lors de la rédaction d'un email
2. Cliquer sur **Joindre** ou l'icône trombone
3. Sélectionner le fichier à joindre
4. Attendre l'upload
5. Envoyer l'email

---

### Créer un dossier

1. Clic droit sur **Boîte de réception**
2. Cliquer sur **Créer un nouveau dossier**
3. Entrer le nom du dossier
4. Cliquer sur **OK**

---

### Déplacer un email vers un dossier

1. Sélectionner l'email
2. Clic droit → **Déplacer vers**
3. Sélectionner le dossier de destination

---

### Configurer sa signature

1. Cliquer sur **Paramètres** (icône engrenage)
2. Aller dans **Identités**
3. Cliquer sur votre identité
4. Remplir le champ **Signature**
5. Cliquer sur **Enregistrer**

---

### Utiliser Thunderbird

1. Ouvrir Thunderbird
2. L'interface affiche les dossiers à gauche, les emails au centre
3. Utilisation similaire au webmail :
   - **Écrire** : nouveau message
   - **Répondre** : répondre à un email
   - **Supprimer** : supprimer un email

---

## FAQ

### Je n'arrive pas à me connecter
- Vérifier l'adresse email (format prenom.nom@tssr.lan)
- Vérifier le mot de passe
- Vérifier que le Caps Lock n'est pas activé

### Je ne reçois pas les emails
- Vérifier le dossier **Spam** / **Indésirables**
- Vérifier que l'expéditeur a bien envoyé le mail
- Contacter l'administrateur

### La pièce jointe est trop volumineuse
- La taille maximale est généralement de 25 Mo
- Utiliser un service de partage de fichiers pour les gros fichiers

### Comment changer mon mot de passe ?
- Aller dans **Paramètres** → **Mot de passe** (si disponible)
- Ou contacter l'administrateur
