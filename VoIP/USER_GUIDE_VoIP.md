
## Utilisation de base

### Passer un appel

1. Ouvrir le softphone 3CX
2. Vérifier que le statut est **vert** (Registered)
3. Composer le numéro de l'extension (ex : 1002)
4. Cliquer sur l'icône d'appel (téléphone vert)
5. Attendre que le correspondant réponde

---

### Recevoir un appel

1. Quand le téléphone sonne, une notification apparaît
2. Cliquer sur **Answer** (décrocher) pour répondre
3. Ou cliquer sur **Decline** (raccrocher) pour refuser

---

### Mettre fin à un appel

1. Pendant un appel
2. Cliquer sur l'icône de raccrocher (téléphone rouge)

---

### Mettre un appel en attente

1. Pendant un appel
2. Cliquer sur **Hold**
3. Pour reprendre, cliquer sur **Resume**

---

## Utilisation avancée

### Transférer un appel

1. Pendant un appel
2. Cliquer sur **Transfer**
3. Composer le numéro de destination
4. Appuyer sur Entrée pour transférer

---

### Faire une conférence

1. Pendant un appel avec un premier correspondant
2. Cliquer sur **Conference**
3. Composer le numéro du troisième participant
4. Une fois connecté, cliquer sur **Merge** pour fusionner

---

### Consulter l'historique des appels

1. Dans 3CX
2. Cliquer sur l'onglet **History**
3. Voir la liste des appels émis/reçus

---

### Régler le volume

1. Pendant un appel ou en veille
2. Utiliser les contrôles de volume du softphone
3. Ou utiliser les paramètres audio de Windows

---

## FAQ

### Le softphone affiche "Not Registered" (rouge)
- Vérifier la connexion réseau
- Redémarrer le softphone
- Vérifier les paramètres du compte
- Contacter l'administrateur

### Je n'entends pas mon correspondant
- Vérifier que le casque/haut-parleur est bien connecté
- Vérifier le volume Windows
- Vérifier le volume dans le softphone
- Essayer avec un autre périphérique audio

### Mon correspondant ne m'entend pas
- Vérifier que le microphone n'est pas coupé (mute)
- Vérifier les paramètres audio dans 3CX
- Vérifier les permissions microphone de Windows

### L'appel se coupe
- Vérifier la qualité de la connexion réseau
- Contacter l'administrateur si le problème persiste
